# 📐 Mobile UX Guidelines

## Spacing
- Prefer spacing over borders
- Keep layout clean and airy

## Typography
- Names: 32–44px (or larger)
- Section titles: 16–18px
- Body text: 13–15px

## Layout
- Center-aligned for elegance
- Full-width sections
- Avoid clutter

## Interaction
- Large tap targets
- Smooth scroll experience

---

# ✨ Micro-Interactions

- Fade-in for names
- Scroll-based reveal for sections
- Subtle parallax for decorative elements
- Smooth transitions
